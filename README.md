## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/REISUB/REISUB_Meeting.git
```

Go to the project directory:

```bash
cd REISUB_Meeting
```

Install dependencies:

```bash
flutter pub get
```

Run the application:

```bash
flutter run
```
